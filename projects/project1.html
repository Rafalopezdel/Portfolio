GifExpertApp

React

Esta aplicación web desarrollada con React permite a los usuarios realizar búsquedas de GIFs animados mediante una interfaz sencilla e intuitiva. Al ingresar un término en el campo de búsqueda, la app consulta una API de imágenes animadas (Giphy) y muestra los 10 primeros resultados de forma dinámica, sin necesidad de recargar la página. Los GIFs se presentan en un diseño responsivo y optimizado para ofrecer una buena experiencia tanto en escritorio como en dispositivos móviles.

Además del buscador, la aplicación mantiene un historial de las últimas 3 búsquedas recientes realizadas por el usuario. Cada vez que se realiza una nueva búsqueda, los resultados correspondientes se agregan a la vista principal, mostrando de forma acumulativa los GIFs de hasta 3 búsquedas distintas, permitiendo al usuario acceder rápidamente a búsquedas recientes sin tener que reescribirlas.

Como parte de las buenas prácticas de desarrollo, la aplicación cuenta con una suite de pruebas automatizadas desarrolladas con Jest, que aseguran el correcto funcionamiento de los componentes principales: el input de búsqueda, la renderización de resultados, y la actualización del historial. Las pruebas permiten mantener la calidad del código y facilitar futuras mejoras o correcciones. En conjunto, esta aplicación demuestra el uso eficiente de React para construir interfaces interactivas, junto con el enfoque en la experiencia del usuario y la estabilidad del software.
