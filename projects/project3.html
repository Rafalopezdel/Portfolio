AuthApp

Angular - Nest

La aplicación de autenticación se compone de un frontend desarrollado en Angular y un backend construido con NestJS, utilizando MongoDB como sistema de base de datos. El módulo permite a los usuarios registrarse y acceder mediante un sistema de login protegido. La interfaz en Angular está diseñada de manera intuitiva, brindando formularios claros para el registro y el inicio de sesión, con validaciones tanto en tiempo real como al enviar los datos. Además, gestiona los estados de autenticación para mostrar o restringir vistas según si el usuario está logueado o no.

En el backend, NestJS implementa un conjunto de controladores, servicios y esquemas que permiten manejar el ciclo completo del usuario: desde su creación hasta su autenticación. Las contraseñas se almacenan de forma segura utilizando hashing (por ejemplo con bcrypt), y se implementan tokens JWT para mantener sesiones seguras. El backend también ofrece un CRUD completo sobre los usuarios, lo que permite no solo registrar, sino también consultar, actualizar o eliminar usuarios desde la base de datos MongoDB.

El sistema de autenticación se comunica entre frontend y backend a través de peticiones HTTP utilizando HttpClient en Angular, y rutas protegidas con guards para asegurar la navegación. Gracias a la arquitectura modular de NestJS y al modelo NoSQL flexible de MongoDB, el sistema es fácilmente escalable y mantenible, ideal para integrarse como base en proyectos más grandes o como microservicio autónomo de autenticación.
