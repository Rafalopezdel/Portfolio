CalendarApp

MERN Stack

El proyecto Calendar App es una aplicación web desarrollada con el objetivo de permitir a los usuarios gestionar sus eventos personales mediante un calendario interactivo. En el frontend, construido con React, se integraron múltiples herramientas modernas como React Router DOM para el manejo eficiente de rutas, Bootstrap para una interfaz responsiva y agradable, y Font Awesome para el uso de iconografía clara y elegante. La visualización de los eventos se logró gracias a la librería react-big-calendar, que permite ver los eventos organizados por días, semanas o meses. Se empleó react-modal para mostrar formularios en ventanas emergentes, react-date-picker para facilitar la selección de fechas, y sweetalert2 para mostrar alertas informativas y confirmaciones de acciones. La lógica de estado global de la aplicación fue gestionada con React Redux y Redux Toolkit, simplificando la manipulación del estado de autenticación y los eventos del calendario

En cuanto al backend, fue desarrollado con Express sobre Node.js, siguiendo una estructura modular y escalable. Se conectó con una base de datos en la nube utilizando MongoDB Atlas, y se utilizó Mongoose para modelar los datos y realizar operaciones CRUD sobre los eventos y usuarios. La autenticación de usuarios fue implementada mediante JWT (JSON Web Tokens), permitiendo un sistema seguro de login y control de sesiones. Se aplicó middleware para la protección de rutas privadas y validación de datos con express-validator. El backend también incluye endpoints RESTful bien definidos para la creación, actualización, eliminación y consulta de eventos, así como para el registro y login de usuarios.

Ambos entornos están completamente integrados, permitiendo que el frontend consuma los servicios del backend de forma fluida a través de peticiones HTTP con axios. Además, se incluyeron pruebas unitarias en el frontend con Jest y @testing-library/react, asegurando que los componentes más importantes funcionen correctamente. El proyecto fue pensado tanto para un entorno de desarrollo local como para su despliegue en plataformas como Railway (backend) y servicios estáticos como Netlify o Vercel (frontend). En conjunto, este sistema de calendario es una solución completa, moderna y funcional, desarrollada con tecnologías de vanguardia del ecosistema JavaScript.
