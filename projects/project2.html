PlayJavaScriptApp

JavaScript

Esta aplicacion pone a prueba varias funcionalidades desarrolladas en javaScript aplicadas para desarrollar juegos y herramientas, empezando con el juego clásico de Snake desarrollado en JavaScript puro, utilizando HTML y CSS para la estructura visual. El tablero se crea dinámicamente en el DOM y está compuesto por una cuadrícula de 10x10. La serpiente se representa como un arreglo de posiciones y se mueve automáticamente en intervalos regulares, respondiendo a las teclas de flechas para cambiar de dirección. El juego incluye lógica para detectar colisiones con los bordes, consigo misma y para gestionar el crecimiento al comer alimentos, los cuales se generan aleatoriamente en las casillas vacías. El sistema de puntuación se actualiza en tiempo real y se muestra en pantalla, y al perder, se detiene el juego y se muestra una señal de "Game Over". Entre sus funcionalidades representativas destacan el uso de estructuras bidimensionales para modelar el tablero, un sistema eficiente de control de colisiones y una interfaz limpia e interactiva sin el uso de librerías externas.

Tambien esta incluida una aplicación de calculadora que permite realizar operaciones matemáticas básicas como suma, resta, multiplicación y división, mediante una interfaz limpia e intuitiva. El proyecto se basa en una estructura orientada a objetos, con una clase Calculadora que define las operaciones aritméticas, y una clase Display que gestiona la interacción entre los botones y la pantalla de resultados. El usuario puede ingresar números, elegir operaciones, borrar caracteres o reiniciar la operación completa. La aplicación incluye control de errores básicos, como la prevención de múltiples puntos decimales o el intento de calcular sin valores válidos. Todo el flujo de la lógica está encapsulado en clases, lo cual facilita su mantenimiento, escalabilidad y organización del código, sin necesidad de librerías externas ni frameworks.

Se puede ver ademas una aplicación de control de tareas que permite a los usuarios gestionar su lista de pendientes de manera sencilla e interactiva. Al cargar la página, se muestra la fecha actual desglosada en día, mes, año y día de la semana, lo que brinda un contexto útil para organizar las tareas. Los usuarios pueden añadir nuevas tareas mediante un formulario, y estas se insertan dinámicamente en el contenedor principal. Cada tarea puede marcarse como completada al hacer clic sobre ella, aplicando una clase visual que indica su estado. Además, la aplicación cuenta con una función para ordenar las tareas, mostrando primero las pendientes y luego las completadas, lo que mejora la productividad y claridad visual. Todo esto se logra sin el uso de frameworks, utilizando solo HTML, CSS y JavaScript nativo para una solución ligera y funcional.

Tambien podemos ver otras aplicaciones como un cronometro, un cifrador cesar y un juego de ahorcado.
