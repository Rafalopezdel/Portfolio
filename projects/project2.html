Project Overview

CalendarioApp is a full-stack collaborative calendar application built with the MERN Stack (MongoDB, Express, React, Node.js). The application features secure user authentication with JWT tokens, allowing multiple users to create, manage, and share calendar events in real-time.

The project demonstrates modern web development practices including RESTful API design, state management with Redux Toolkit, protected routes, and a responsive user interface that adapts seamlessly across all devices.

The application includes dynamic routing with React Router, smooth animations, interactive components, contact form integration, and performance optimizations including code splitting and lazy loading for an optimal user experience.
